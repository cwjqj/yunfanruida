# Radar_Top_Track_2.0 极限与三人连续性验证

测试日期：2026-07-15  
测试性质：离线构造帧与回归测试

## 1. 结论

- 23 条目标的单帧数据区为 `23 × 11 = 253` 字节，低于脚本 256 字节限制；连续 4 帧后 23 条均被解析并确认为稳定航迹。
- 云端只输出 `track0`、`track1`、`track2` 三条完整对象，内部仍统计全部 23 条。
- 三人测试中，雷达原始 ID 从 `1/2/3` 变为 `7/8/9`，短时丢失后又变为 `10/11/12`；软件 UID 始终保持 `1/2/3`。
- 三条轨迹均保留非空路径，累计位移均不少于 60cm，恢复后状态为 active，并产生 revived 事件。
- 以上只验证软件边界，不代表雷达硬件可真实同时分离 23 人。
