# 云帆瑞达 R60BMP1 雷达设置指南

> 基于用户手册 v1.5 | 产品：60G毫米波雷达 R60BMP1 人数统计雷达

## 一、基本串口参数

| 参数 | 值 |
|------|-----|
| 接口电平 | TTL (3.3V) |
| 波特率 | 115200 bps |
| 停止位 | 1 |
| 数据位 | 8 |
| 奇偶校验 | 无 |

## 二、硬件设置要求

### 2.1 电源要求

| 参数 | 要求 |
|------|------|
| 供电电压 | 4.9 - 6V |
| 额定电流 | ≥ 200mA |
| 电源纹波 | ≤ 100mV |

### 2.2 天线与外壳布局

| 项目 | 要求 |
|------|------|
| PCBA贴件高度 | 雷达贴件高度比其他器件 ≥ 1mm |
| 天线面与外壳距离 | 2 - 5mm |
| 外壳材质 | 非金属，探测面需平直 |

### 2.3 静电防护

- 避免触摸雷达模块天线表面和连接器管脚
- 操作时佩戴防静电手套

## 三、功能概述

| 功能点 | 说明 |
|--------|------|
| DP1：有人/无人 | 无人→有人：0.5s内上报；有人→无人：约30s上报 |
| DP2：人数统计 | 探测区域内人员实时检测 |
| DP3：轨迹信息 | 上报目标位置、高度、速度信息 |

---

## 四、帧结构定义

```
帧头(2B) | 控制字(1B) | 命令字(1B) | 长度标识(2B) | 数据(nB) | 校验码(1B) | 帧尾(2B)
0x53 0x59 |   Control  |  Command  | Len_H Len_L  |  Data...  |    sum     | 0x54 0x43
```

**校验码计算**：帧头 + 控制字 + 命令字 + 长度标识 + 数据，求和后取低8位

---

## 五、控制字定义

| 控制字 | 含义 |
|--------|------|
| `0x01` | 心跳包标识 |
| `0x02` | 产品信息 |
| `0x03` | OTA升级 |
| `0x05` | 工作状态 |
| `0x06` | 安装方式 |
| `0x07` | 人数统计配置 |
| `0x80` | 人体存在 |
| `0x82` | 轨迹跟踪 |
| `0x86` | 人数统计 |

## 六、关键设置项详解

### 6.1 安装角度设置（控制字 `0x06`，命令字 `0x01`）

**设置指令**：`53 59 06 01 00 06 [X轴角度2B] [Y轴角度2B] [Z轴角度2B] sum 54 43`

| 参数 | 说明 |
|------|------|
| X轴角度 | 默认0° |
| Y轴角度 | 默认0° |
| Z轴角度 | 默认下倾30° |
| 角度范围 | 实际 -180° ~ 180°；协议值扩大100倍为 -18000 ~ 18000 |
| 支持模式 | 30°（侧装）、90°（顶装） |

**注意**：设置安装角度后会自动对应增加标签门、变更探测范围、变更安装高度

**查询指令**：`53 59 06 81 00 01 0F sum 54 43`

---

### 6.2 安装高度设置（控制字 `0x06`，命令字 `0x02`）

**设置指令**：`53 59 06 02 00 02 [高度信息2B] sum 54 43`

| 参数 | 说明 |
|------|------|
| 单位 | cm |
| 步长 | 1cm |

**查询指令**：`53 59 06 82 00 01 0F sum 54 43`

### 6.8 雷达探测范围设置（控制字 `0x07`，命令字 `0x09`）

**设置指令**：`53 59 07 09 00 09 00 [x正半轴2B] [x负半轴2B] [y正半轴2B] [y负半轴2B] sum 54 43`

| 参数 | 说明 |
|------|------|

| x负半轴 | 雷达后方探测距离 |
| y正半轴 | 雷达右侧探测距离 |

**查询指令**：`53 59 07 89 00 01 0F sum 54 43`

---

### 6.9 标签设置（控制字 `0x07`，命令字 `0x11`）

**设置指令**：`53 59 07 11 00 06 [标签类型1B] [范围类型1B] [X2B] [Y2B] sum 54 43`

**标签类型**：

| 值范围 | 含义 |
|--------|------|
| 0 - 4 | 门（人数进出统计） |
| 5 - 9 | 沙发（靠近/远离） |
| 10 - 14 | 床（靠近/远离） |
| 15 - 19 | 干扰源（靠近/远离） |
| 20 | 卫生间门（靠近/远离） |
| 21 - 30 | 保留 |

**范围类型**：

| 值 | 含义 |
|----|------|
| `0` | 圆形（X=半径，Y不用） |
| `1` | 矩形（X=宽，Y=高） |

**查询指令**：`53 59 07 91 00 01 0F sum 54 43`

---

### 6.10 标签清除（控制字 `0x07`，命令字 `0x13`）

**设置指令**：`53 59 07 13 00 01 [0xFF/标签索引] sum 54 43`

| 数据值 | 含义 |
|--------|------|
| `0xFF` | 清除全部标签 |
| 标签索引 | 清除指定索引的标签 |

---

### 6.11 干扰边设置（控制字 `0x07`，命令字 `0x1A`）

**设置指令**：`53 59 07 1A 00 01 [1/2/3/4] sum 54 43`

| 数据值 | 含义 |
|--------|------|
| `1` | 左侧0.5米边缘，前方探测距离4米 |
| `2` | 右侧0.5米边缘，前方探测距离4米 |
| `3` | 酒店走廊，左右各0.75米 |
| `4` | 没有边界设置 |

> **注意**：仅适用于门上方中央倾斜30°安装模式

**查询指令**：`53 59 07 9A 00 01 0F sum 54 43`

## 七、常用设置流程

### 7.1 雷达上电初始化流程

1. 等待雷达主动上报初始化完成信息（`53 59 05 01 00 01 0F sum 54 43`）
2. 或主动查询初始化状态：`53 59 05 81 00 01 0F sum 54 43`
3. 确认初始化完成后，进行各项设置

### 7.2 安装配置流程

```
1. 设置安装角度 → 53 59 06 01 00 06 [X轴] [Y轴] [Z轴] sum 54 43
2. 设置安装高度 → 53 59 06 02 00 02 [高度] sum 54 43
3. 设置干扰边 → 53 59 07 1A 00 01 [模式] sum 54 43
4. 设置标签门 → 53 59 07 11 00 06 [类型] [范围] [X] [Y] sum 54 43
```

### 7.3 功能启用流程

```
1. 开启人体存在 → 53 59 80 00 00 01 01 sum 54 43
2. 开启轨迹跟踪 → 53 59 82 00 00 01 01 sum 54 43
3. 设置实时人数上报间隔 → 53 59 86 0B 00 04 [时间] sum 54 43
4. 设置精准人数上报间隔 → 53 59 86 0D 00 04 [时间] sum 54 43
```

## 八、参数设置示例

### 示例1：设置顶装模式（90°下倾）

```
设置安装角度: 53 59 06 01 00 06 00 00 00 00 23 28 04 54 43
  - X轴: 0° → 00 00
  - Y轴: 0° → 00 00
  - Z轴: 90° × 100 = 9000 → 23 28
  - 校验码: 04
```

> 更正说明：`03 84` 是十进制 900，按扩大100倍传输的规则仅表示 9.00°；顶装 90°必须使用 `23 28`。

### 示例2：设置安装高度为2.5米

```
设置安装高度: 53 59 06 02 00 02 09 C4 sum 54 43
  - 2.5米 = 250cm → 0x09C4
```

### 示例3：设置标签门（矩形，宽100cm，高50cm）

```
设置标签门: 53 59 07 11 00 06 00 01 00 64 00 32 sum 54 43
  - 标签类型: 00 (门)
  - 范围类型: 01 (矩形)
  - 宽: 100cm → 00 64
  - 高: 50cm → 00 32
```

### 示例4：开启人体存在和轨迹跟踪

```
开启人体存在: 53 59 80 00 00 01 01 sum 54 43
开启轨迹跟踪: 53 59 82 00 00 01 01 sum 54 43
```

## 九、校验码计算示例

**存在信息查询指令**：

```
帧头:     0x53 0x59
控制字:   0x80
命令字:   0x81
长度标识: 0x00 0x01
数据:     0x0F

校验码 = (0x53 + 0x59 + 0x80 + 0x81 + 0x00 + 0x01 + 0x0F) & 0xFF
       = 0x01BD & 0xFF
       = 0xBD

完整指令: 53 59 80 81 00 01 0F BD 54 43
```
