# Posture 3.0 三人姿态离线验证

版本：`Radar_Top_Posture_3.0`  
日期：2026-07-15

## 验证范围

- 三个稳定 UID 同时输出站、坐、躺。
- 雷达原始 ID 跳变后姿态仍绑定原 UID。
- 单帧低高度尖刺不把站姿切换为躺姿。
- 航迹短时丢失进入 holding 时保留姿态码并使 `posture_valid=0`。
- revived 后恢复有效，三个姿态不串人。
- 站/坐和坐/躺阈值附近抖动不造成反复切换。
- 持续高度变化能够按顺序确认站、坐、躺、重新站立。
